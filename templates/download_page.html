Getting started

A vpn connection is required to connect to the challenge containers

Download the OpenVPN client Generate a VPN configuration Connect to the VPN Test your connection
The vpn client can be downloaded from here: openvpn.net/download-open-vpn/
If you are using linux you should download it as a package
Each vpn client requires a unique configuration file. If you have already generated one, please download it again instead of geneating a new one


If you need to revoke the access rights to all previous config files press this button
If you are using linux open a terminal in the same directory as your config file, then run the command:
sudo openvpn <username>.ovpn

For Windows users you can follow this guide: imgur
Go to this link, if you see "VPN CONNECTION WORKS!" the vpn connection works
